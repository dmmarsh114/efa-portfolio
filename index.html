About Me

I graduated from Indiana University in 2016 with a desire to work outdoors, see other places, and give back to the world. I volunteered for 2 months doing hurricane disaster relief in the U.S. Virgin Islands, then did much longer stint with the Montana Conservation Corps in the remote wilderness. Along the way, I worked in landscaping and as a grower in a commercial greenhouse in the Indianapolis area. During that time, I began to teach myself how to code. I taught myself the basics of the C# programming language by building simple games using the Unity Engine.

At first programming was just a hobby of mine, but I quickly realized that I was even more interested in working with computers than I was with plants. I began looking for ways to further my education in programming, with an eye to a future career. That’s when I discovered the Eleven Fifty Academy -- a coding bootcamp in Indianapolis. Now that I’m enrolled, I am expanding my knowledge of HTML, CSS, and Javascript. After this, I’ll be entering the job market, and I’m sure I’ll always be learning more, designing more, and enhancing my coding skills.

CSS Creature

This was one of the first projects that I made while I was at the Eleven Fifty Academy. I got to experiment with basic HTML tags, as well as CSS classes and IDs. I learned how to use CSS for styling HTML elements through the use of borders, backgrounds, and colors.

I also learned how to use CSS to give the creature some simple animations, and I was able to apply those practices to future projects. Above all else, I learned a great deal about how to place and arrange elements using CSS. I struggled with adjusting the display and position settings at first, but by the end of the project I was starting to feel comfortable with the basic concepts.

EFA Gold Badge

Static Layout

The Static Layout was a more robust exercise in constructing HTML pages and styling them with CSS. The goal here was to recreate the landing page for Hulu’s main website. Aside from the background image, I feel like I was able to capture the overall style pretty well.

This project was also a lesson in organizing my code: because I had more elements and sections to manage than I did with the CSS Creature, I had to be very mindful of how I was naming and using my classes and IDs.

There were more than a few interesting problems that I had to solve while making this layout, but chief among them was the navigation bar. Hulu’s website had a navbar that was fixed to the top of the screen, but its background color would change as the user scrolled down the page. In order to recreate this effect, I used some basic DOM manipulation to add and remove classes to the navbar, thereby giving it the ability to change its style dynamically.

API Challenge
